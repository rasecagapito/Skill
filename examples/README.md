## Learning from Examples

You can **learn** from these scripts:

### Example 1: How to measure freed space

```powershell
$sizeBefore = (Get-ChildItem -Path $cachePath -Recurse | 
               Measure-Object -Property Length -Sum).Sum / 1GB

# [cleanup happens here]

$sizeAfter = (Get-ChildItem -Path $cachePath -Recurse | 
              Measure-Object -Property Length -Sum).Sum / 1GB

$liberado = [math]::Round($sizeBefore - $sizeAfter, 2)
Write-Host "Liberado: $liberado GB" -ForegroundColor Green
```

### Example 2: Safe error handling

```powershell
Remove-Item "$cachePath\*" -Recurse -Force -ErrorAction SilentlyContinue
```

The `-ErrorAction SilentlyContinue` prevents the script from crashing if a file is locked or doesn't exist.

### Example 3: Service control

```powershell
# Stop service
Stop-Process -Name "GoogleDriveFS" -Force -ErrorAction SilentlyContinue
Start-Sleep -Seconds 3

# [cleanup happens here]

# Restart service
Start-Process "C:\Program Files\Google\Drive File Stream\GoogleDriveFS.exe"
```
